# blinkenlights-badge
Show blinkenlights movies on a "LED name badge"

![LED badge](ledman_c.gif)

## Play direct BML movies 
Play the old movies of the famous "Blinkenlights Project" of the CCC.
Or create your own.
The limitations are 
- only 123 frames 
- no grey shades
